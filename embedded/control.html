1. Open loop, Closed Loop

Open loop

  • Feedforward action. Control action is based on knowledge to be controlled (called plant)
  • No feedback input, only output
  • Requires accurate model to be effective (a lot of fine tuning)
  • Cannot compensate changes that is added to controller device
  • Basically, No sensors. For example: Toaster

    • Closed loop

  • Feedback action
  • Measurements of plant output are fed back to the controller
  • Generally superior performance (accuracy, bandwidth, disturbance rejection) compared to open loop controlled
  • Has tendency towards instability
  • Has sensors (output transducer) that measures output response
  • Compensate changes by measuring output via sensors
  • Transient response and steady state error for easier control with flexibility of closed loop system (by adjusting amplification)
  • Actuating signal is when output is subtracted from input signal. If both input and output transducers have unity gain(amplifies input by 1), actuating signal value is equal to actual difference between input and output (error)

    • There are 2 goals when designing a control system. First is achieve Transient Response and is Steady-State Response that decays to zero fast

    2. Stability

    Taking lift as an example

    A. Transient Response

  • Slow transient is when changes is too slow. EG: Users grow impatient
  • Fast transient is when changes is too fast. EG: Users feel uncomfortable. Fast transient response may cause physical damage to components. Fast transient comes with consequent.
  • Sum of natural and forced but natural response is large

    • B. Steady-State Response

  • Resembles input and is remained after transient has decayed to 0. EG: Lift reaches the floor exactly at the space of lift's door
  • Concerned about the accuracy of steady-state response (steady-state error).
  • Sum of natural and forced but natural response is small

    • C. Stability

    Natural Response

  • The way the system dissipates or acquires energy
  • Is dependent on system and independent of input
  • EG: Capacitor charged to some initial voltage by an external circuit, which is disconnected awhile ago

    • Forced Response

  • 2 different definitions
  • Circuit's final steady state (Agarwal&Lang, 2005)
  • Dependent on input
  • EG: Stop voltage as step input.

    • Total Response

    Total Response = Transient response temporary part (natural response) + Steady-state response permanent part (forced response)

    3. Design Process

    1. Transform requirements into physicla system and specification from requirements
    2. Draw functional block diagram
    3. Transform physical system into schematic
    4. Use schematic to obtain block diagram, signal-flow diagram, or state-space representation
    5. Reduce block diagram to a single-block or closed-loop system
    6. Analyze, design and test if requirement and specification are met

    A. Transform requirement into physical system

  • Desired position? Desired features? Weight? Physical dimensions?
  • These are determined by the organizer during a robotic competition

    • B. Block Diagrams

    Translates from step 1 into a diagram that describes the component parts (function and/or hardware). It indicates the functions such as input sensors, motor drivers, controllers, hardware description, to produce a detailed layout of the system.

    F. Analyze and Design

    Physical laws such as Kirchoff's laws for electrical networks and Newton's law for mechanical system for mathamatical model development(block diagram)

    The steps of block making until prior to this step can be reduced if only the performance of a single subsystem is interested (EG:Only encoder and wheel)

  • Test input signal for both analytical and during testing are used to verify the design
  • It is important to choose simple and standarized test inputs to avoid complications
  • Such inputs are impulses, steps, ramps, parabolas and sinusoids

    • Impulse input

  • Is infinite at t=0 and 0 everywhere else
  • Area under the unit impulse is 1 where the approximation of this type of waveform is used to place initial energy into system so that the response due to that initial energy is only a transient response of a system
  • This response is used to derive a mathematical model of the system

    • Step input

  • Represents a constant command
  • EG: position, velocity, acceleration
  • Typically step input command is of the same form of the output
